Create Dax Measures 12

Creating the following measure by typing in the DAX calculation click a new measure and write the DAX formula, here is the DAX formula what you use for this dashboard. You can copy and paste.

  • Avenged Dominance Score = AVERAGE(DISC_ClassScores[Dominance])
  • Avenged Influencer Score = AVERAGE(DISC_ClassScores[Influencer])
  • Average Steadiness Score = AVERAGE(DISC_ClassScores[Steadiness])
  • Average Compliance Score = AVERAGE(DISC_ClassScores[Compliance])
  • Total Student = COUNT(DISC_ClassScores[UserID])

 

  • Class Batch = VAR StringPart =  (SEARCH("-", DISC_ClassScores [ClassName])) VAR StringText = RIGHT(DISC_ClassScores[ClassName], LEN(DISC_ClassScores[ClassName]) - StringPart) RETURN StringText
  • ClasssCompletionStatus = IF(DISC_ClassScores[ClassCompleted] = 1, "ClassCompleted","ClassInComplete")

 

  • Success Rate = AVERAGE( (DISC_ClassScores[ClassCompleted]))
